# Archive 目录说明

此目录用于存放项目的旧版本文件和已弃用的代码，保留这些文件仅供参考和历史记录使用。

## 文件列表

- **main.py** - 旧版主程序，基于Tkinter的GUI界面
- **basic_reader.py** - 基础阅读器实现
- **simple_reader.py** / **simple_reader_fixed.py** - 简化版阅读器
- **main_simple.py** / **main_simple_fixed.py** - 简化版主程序
- **README_PROJECT.md** - 旧版项目说明文档
- **novel_reader.log** - 历史日志文件

## 当前活跃版本

当前项目使用基于PyQt5的浏览器版本（browser.py），提供更强大的功能：
- 完整的浏览器功能
- 智能内容提取
- OCR识别支持
- AI智能总结

如需使用最新版本，请查看主目录的README.md文档。
