# 小说阅读神器项目总结

## 项目概述

小说阅读神器是一款帮助用户快速浏览和理解小说内容的工具。该项目提供了多个版本以适应不同的使用需求：

1. **完整版**（main.py）：包含 PaddleOCR 图片文字识别和 AI 文本总结功能
2. **简化版**（simple_reader.py）：专注于文本总结，使用 jieba 分词
3. **基础版**（basic_reader.py）：最简单的文本分析工具，无需额外依赖

## 项目文件结构

```
novel_reader_helper/
├── main.py                      # 完整版主程序（含OCR功能）
├── simple_reader.py             # 简化版（使用jieba分词）
├── basic_reader.py              # 基础版（最小依赖）✅ 推荐使用
├── ai_summary.py                # AI总结模块
├── install.py                   # 完整版安装脚本
├── simple_install.py            # 简化版安装脚本
├── start.py                     # 启动脚本
├── requirements.txt             # 依赖包列表
├── README.md                    # 项目说明文档
├── INSTALL_GUIDE.md            # 详细安装指南
└── README_PROJECT.md           # 本文档
```

## 快速开始

### 方式一：使用基础版（推荐）

基础版不需要安装额外依赖，可以直接运行：

```bash
cd novel_reader_helper
python basic_reader.py
```

**功能特点：**
- ✅ 打开并分析文本文件
- ✅ 统计字符数、行数、段落数
- ✅ 估算阅读时间
- ✅ 提取开头和结尾内容
- ✅ 保存分析结果

### 方式二：使用简化版

如果您需要更智能的分析功能，可以使用简化版：

```bash
# 1. 安装依赖
python simple_install.py

# 2. 运行程序
python simple_reader.py
```

**功能特点：**
- ✅ 基础版的所有功能
- ✅ 智能提取关键词
- ✅ 自动识别重要句子
- ✅ 生成内容摘要

### 方式三：使用完整版

如果您需要从图片中识别文字，可以使用完整版：

```bash
# 1. 安装依赖（可能需要较长时间）
python install.py

# 2. 运行程序
python main.py
```

**功能特点：**
- ✅ 简化版的所有功能
- ✅ 图片文字识别（OCR）
- ✅ 支持多种图片格式

## 使用说明

### 基础版使用方法

1. **启动程序**
   ```bash
   python basic_reader.py
   ```

2. **打开文本文件**
   - 点击"📂 打开文本文件"按钮
   - 选择您要分析的小说文本文件

3. **进行分析**
   - 点击"📊 简单分析"按钮
   - 查看右侧的分析结果

4. **保存结果**
   - 点击"💾 保存结果"按钮
   - 选择保存位置

### 简化版使用方法

1. **启动程序**
   ```bash
   python simple_reader.py
   ```

2. **输入文本**
   - 方式一：点击"?? 打开文本文件"加载文件
   - 方式二：点击"📄 输入文本内容"直接粘贴文本

3. **调整参数**
   - 关键词数量：3-20个（默认8个）
   - 摘要句子数量：2-10个（默认5个）

4. **生成总结**
   - 点击"📝 AI总结"按钮
   - 等待处理完成
   - 查看分析结果

5. **保存结果**
   - 点击"💾 保存结果"保存到文件

## 核心功能说明

### 1. 文本统计分析
- 字符数统计
- 中文汉字统计
- 行数和段落数统计
- 阅读时间估算

### 2. 内容提取
- 自动提取章节标题
- 提取开头和结尾内容
- 识别重要段落

### 3. 关键词提取（简化版及以上）
- 基于 TF-IDF 算法
- 自动识别文本主题
- 可调节关键词数量

### 4. 智能摘要（简化版及以上）
- 提取重要句子
- 按重要性排序
- 保持原文顺序

### 5. OCR文字识别（完整版）
- 支持多种图片格式
- 识别图片中的文字
- 转换为可编辑文本

## 常见问题

### Q1: 哪个版本适合我？

- **基础版**：适合只需要简单文本分析的用户，无需安装额外依赖
- **简化版**：适合需要智能关键词和摘要的用户，依赖少
- **完整版**：适合需要OCR功能的用户，依赖较多

### Q2: 程序无法启动怎么办？

1. 检查 Python 版本（需要 3.7+）
2. 确认依赖包已安装
3. 尝试使用基础版

### Q3: 如何提高分析质量？

1. 使用简化版或完整版
2. 调整关键词和摘要数量参数
3. 确保输入文本格式正确

### Q4: 支持哪些文件格式？

- 文本输入：支持 .txt 文件
- 图片识别（完整版）：支持 .png, .jpg, .jpeg, .bmp 等
