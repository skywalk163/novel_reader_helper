# PaddleOCR Integration

This directory contains the PaddleOCR integration for the Novel Reader Helper project.

Files in this directory:
- app.py: Main OCR service application
- examples.py: Example usage of the OCR service
- generate_test_image.py: Script to generate test images
- requirements.txt: OCR-specific dependencies
- start_service.bat: Batch script to start the OCR service on Windows
- test_ocr.py: OCR testing script
- test_service.py: OCR service testing script
- USAGE.md: Usage instructions for the OCR service

For detailed usage instructions, please refer to USAGE.md in this directory.
